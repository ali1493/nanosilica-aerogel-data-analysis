# Nanosilica Aerogel Data Analysis

This repository contains data generation and analysis tools for PVA-SA-GEL biocomposite films reinforced with nano silica, nano silica aerogel, and Zn-doped nano silica aerogel.

## Project Context

The project focuses on environmentally friendly biocomposite films for potential food packaging applications. The films are based on poly(vinyl alcohol), sodium alginate, and gelatin, with different inorganic fillers.

## Sample Groups

- Sample 1: PVA-SA-GEL control
- Samples 2–4: nSiO2-added PVA-SA-GEL
- Samples 5–7: nano silica aerogel-added PVA-SA-GEL
- Samples 8–10: Zn-doped nano silica aerogel-added PVA-SA-GEL

## Planned Analyses

- FTIR peak identification
- XRD crystallinity comparison
- SEM-EDS morphology and elemental distribution
- TGA/DTA/DTG thermal stability comparison
- Synthetic data generation for testing scripts
